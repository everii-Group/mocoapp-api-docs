## GET /account/fixed_costs

Retrieve all fixed costs

```bash
curl -X GET \
  'https://{domain}.mocoapp.com/api/v1/account/fixed_costs' \
  -H 'Authorization: Token token=YOUR_API_KEY'
```

This returns complete fixed cost information (see Attributes).
